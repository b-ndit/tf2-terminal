# TF2 Terminal

A desktop trading terminal for Team Fortress 2 — live market data, trade
analysis, price alerts, and flip-finding, built as a native app (Tauri + Rust
+ React) rather than a browser extension or web dashboard.

> **Disclaimer:** TF2 Terminal is an unofficial, fan-made tool. It is not
> affiliated with, endorsed by, or sponsored by Valve Corporation or
> backpack.tf. Team Fortress 2 is a trademark of Valve Corporation.

## What it does

- **Backpack browser** — virtualized grid for large inventories, tags,
  folders, favorites, notes.
- **Market Analyzer** — paste a backpack.tf classifieds URL to see live
  spread, liquidity/demand scores, buyer/seller tables, and price history
  charts.
- **Trade Analyzer** — ★ rating on incoming Steam trade offers with a
  plain-language explanation and a counteroffer suggestion.
- **Live Feed** — a real-time stream of the backpack.tf listing market.
- **Alerts** — rule-based notifications (price drop, spread widening, new
  buyer/seller, historical high/low) over desktop notifications, Discord
  webhooks, or an in-app sound.
- **Flip Finder** — scans active listings for profitable flips, weighted by
  a confidence score (history depth × liquidity), with a watchlist.

See [`docs/DESIGN.md`](docs/DESIGN.md) for the full architecture, database
schema, and the module-by-module implementation roadmap (this is an
in-progress build — modules 0–11 of 15 are complete; check `CLAUDE.md` for
current status).

## Stack

Tauri 2 (Rust backend, native webview) · React 18 + TypeScript · SQLite
(via `sqlx`) · TanStack Query · Tailwind CSS. All external I/O (Steam Web
API, backpack.tf REST + websocket) lives in the Rust backend; the frontend
only talks to it over typed Tauri commands/events (bindings generated by
`specta`).

## Getting started

Prerequisites: [Rust](https://www.rust-lang.org/tools/install), Node 22+,
and Tauri's [platform-specific
prerequisites](https://tauri.app/start/prerequisites/) (e.g.
`webkit2gtk-4.1` on Linux).

```bash
npm install
npm run tauri dev
```

You'll need your own Steam Web API key
([steamcommunity.com/dev/apikey](https://steamcommunity.com/dev/apikey)) and,
optionally, a backpack.tf API token/token pair — both are entered in-app and
stored in the OS keychain, never in a config file or the database.

## Testing

```bash
cargo test --manifest-path src-tauri/Cargo.toml   # Rust unit/integration tests
npm run test                                       # Vitest (frontend)
npx playwright test                                # E2E (mocks the Tauri IPC bridge)
```

CI (`.github/workflows/ci.yml`) runs `cargo fmt --check`, `cargo clippy -D
warnings`, and the full test suite on every push/PR.
